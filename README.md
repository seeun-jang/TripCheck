# 🧳 TripCheck

## Java 객체지향 프로그래밍을 활용한 여행 준비 관리 프로그램

> **프로그래밍 언어 실습 기말 과제**
> **인공지능 소프트웨어학과**
> **2601110299 장세은**

<br>

## 📌 프로젝트 소개

**TripCheck**는 여행이나 어학연수를 준비할 때 필요한 항목을 관리할 수 있는 Java 콘솔 프로그램입니다.

준비 항목을 **서류**, **짐**, **비용**으로 나누어 추가할 수 있으며, 각 항목의 완료 여부를 확인하고 수정할 수 있습니다.
또한 비용 항목을 입력하면 총 예상 비용과 준비 진행률도 함께 출력됩니다.

<br>

## ✈️ 제작 배경

7월에 약 한 달 동안 세부 어학연수를 갈 예정이어서, 실제 준비 과정에서 사용할 수 있는 체크리스트 프로그램을 만들고자 하였습니다.

여권, 항공권, 옷, 충전기, 생활비 등 준비해야 할 항목을 한눈에 확인하고, 빠뜨리는 물건 없이 준비하기 위해 이 주제를 선택하였습니다.

## 🧩 클래스 구성

| 클래스명               | 역할            |
| ------------------ | ------------- |
| `CheckItem`        | 준비 항목의 부모 클래스 |
| `DocumentItem`     | 서류 항목 클래스     |
| `PackingItem`      | 짐 항목 클래스      |
| `MoneyItem`        | 비용 항목 클래스     |
| `ChecklistManager` | 항목 목록 관리 클래스  |
| `Main`             | 프로그램 실행 클래스   |

<br>

## 💡 객체지향 개념 적용

* **클래스**: 기능별로 클래스를 분리하여 프로그램 구성
* **생성자**: 객체 생성 시 항목 이름과 완료 여부 초기화
* **상속**: `DocumentItem`, `PackingItem`, `MoneyItem`이 `CheckItem` 상속
* **오버라이딩**: 항목 종류에 따라 `getInfo()` 메서드 다르게 출력
* **접근 제한자**: 필드를 `private`으로 선언하여 데이터 보호
* **다형성**: `ArrayList<CheckItem>`으로 여러 자식 객체를 함께 관리

## 🖥️ 실행 화면

### 메인 메뉴

```text
====================================
        TripCheck 여행 준비 관리
====================================
1. 준비 항목 추가
2. 준비 목록 출력
3. 완료 여부 수정
4. 전체 항목 삭제
5. 프로그램 종료
====================================
메뉴 선택:
```

### 준비 목록 출력

```text
[TripCheck 여행 준비 목록]

1. [짐] 모자 - 미완료

-----------------------------
전체 항목 수: 1개
완료 항목 수: 0개
총 예상 비용: 0원
준비 진행률: 0.0%
```
